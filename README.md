# Pseudocode Interpreter

### About the project

An interpreter for a pseudocode-based, dynamic programming language.\
\
It was originally made for a school project, and this is the second version with some improvements added, such as more functionality (functions, strings, etc) and the ability to translate the error messages, names of the built-in stuff, keywords and name of primitive types.\
