# How the files are handled

* The internal Lexer is used.
* Each line should have 2 strings, delimited by two quotes(`"..."`).
* The first should not be modified, as it it the key for what the second string is supposed to represent.
* The second one can be modified to any string you can think of, or leave the quotes empty (`""`) to have the value of the key.
* By default, everything is translated to romanian.
